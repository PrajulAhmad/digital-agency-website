# Digital Agency Website

A professional, modern, and fully responsive website for a digital agency offering Web Design, SEO, and Branding services.

## 📋 Features

### 🎨 Design Features
- **Modern & Professional Design** - Contemporary aesthetic with black and green color scheme
- **Fully Responsive** - Works flawlessly on desktop, tablet, and mobile devices
- **Smooth Animations** - Engaging transitions and hover effects throughout
- **Interactive Elements** - Dynamic user interactions and visual feedback
- **Premium Color Scheme** - Professional black background with vibrant green accents

### 📄 Sections

1. **Navigation Bar** - Sticky header with smooth scrolling navigation and mobile hamburger menu
2. **Hero Section** - Eye-catching landing section with call-to-action button
3. **Services Section** - Showcase three main services:
   - Web Design - Responsive, user-friendly website design
   - SEO - Search engine optimization strategies
   - Branding - Complete brand identity solutions
4. **Portfolio Section** - Display of completed projects with filtering capability
5. **About Section** - Company information with impressive statistics
6. **Revenue Generating Section** - 7-layer framework to convert traffic into sales
7. **CTA Section** - Call-to-action for consultation
8. **Contact Section** - Contact form with validation and contact information
9. **Footer** - Links, social media, and copyright information

### ✨ Interactive Features

- **Mobile Menu** - Hamburger menu for mobile navigation
- **Smooth Scrolling** - Smooth page navigation between sections
- **Form Validation** - Real-time email and field validation
- **Success Notifications** - User feedback for form submissions
- **Scroll-to-Top Button** - Quick navigation to page top
- **Active Link Highlighting** - Navigation highlight shows current section
- **Fade-in Animations** - Element animations on scroll
- **Counter Animations** - Animated statistics counters
- **Portfolio Filtering** - Click tags to filter projects

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No server or build tools required

### Installation

1. **Download or clone the project files**
   - Ensure all three files (index.html, styles.css, script.js) are in the same directory

2. **Open the website**
   - Double-click `index.html` OR
   - Right-click `index.html` → Open with → Your preferred browser

3. **That's it!** The website is ready to use

## 🎯 Customization

### Change Company Name & Logo
Edit `index.html`, line 240:
```html
<h1>Digital<span>Agency</span></h1>
```

### Update Contact Information
Edit `index.html` around line 340:
```html
<p>+1 (555) 123-4567</p>
<p>info@digitalagency.com</p>
<p>123 Business St, Tech City, TC 12345</p>
```

### Modify Colors
Edit `styles.css`, lines 8-14:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    /* ... */
}
```

### Add Your Projects
In `index.html`, add new portfolio items within `.portfolio-grid`:
```html
<div class="portfolio-item">
    <div class="portfolio-image" style="background: linear-gradient(135deg, #color1, #color2);"></div>
    <div class="portfolio-info">
        <h3>Project Name</h3>
        <p>Project Description</p>
        <span class="portfolio-tag">Service Type</span>
    </div>
</div>
```

### Update Services
Modify the service cards in the Services section to match your offerings.

## 🐛 Troubleshooting

### Icons not showing
- Ensure you have an internet connection (Font Awesome loads from CDN)
- Check browser console for errors (F12 → Console)

### Styles not applying
- Ensure styles.css is in the same directory
- Check file path in index.html matches your directory structure

### Mobile menu not working
- Test in browser's mobile view (F12 → Toggle device toolbar)
- Clear browser cache if issues persist
